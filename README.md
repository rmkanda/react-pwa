## Build

- Install Node.js 15.x
- Clone th repo
- Run the commands

  ```
  npm install
  npm run build
  npm run watch
  ```

Tools used

- React
- Webpack 5
- Babel
- ESLint
- Prettier
- PWA (workbox)
